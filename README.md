# 🚀 ServiceNow Support Portal (Frontend Project)

A modern web-based **ServiceNow-like Support Ticket System** built using **HTML, CSS, and JavaScript**.

This project simulates a real-world IT support portal where users can:
- Login to the system
- Create support tickets
- Assign tickets to agents
- Manage users (Admin)
- Track ticket status
- View analytics dashboard

---

# 📌 Features

✅ User Authentication (Login System)  
✅ Ticket Creation & Management  
✅ Automatic Group Assignment  
✅ Agent Assignment System  
✅ Ticket Status (Open / Closed)  
✅ Delete Ticket Function  
✅ Dashboard Statistics  
✅ Charts Dashboard (using Chart.js)  
✅ User Management (Admin Panel)  
✅ Notification System  
✅ LocalStorage Database (No backend required)

# 📊 How It Works

1. User logs into the system  
2. Admin can manage users  
3. User creates a ticket  
4. Ticket is assigned to:
   - **Certificates Team** OR
   - **Platform Team**  
5. Ticket is assigned to an agent  
6. Ticket appears in dashboard  
7. Status can be updated (Open → Closed)  
8. Ticket can also be deleted  
9. Dashboard updates automatically
